ホルムアルデヒドの計算
=======================================================================

まずは一例としてホルムアルデヒドの計算をしてみます。


Gaussianの入力ファイル
-----------------------------------------------------------------------

.. literalinclude:: sample/formaldehyde.gau
   :language: none
   :linenos:


.. warning::

  GaussianはFortranで記述されています。
  計算精度を確保するために、小数点以下がゼロでも小数点を入力してください。


.. warning::

  エラーで停止する場合は、末尾に改行を追加するなど対処してください。


1行目の"%"で始まる行は、Link0コマンドとよばれます。
ここではチェックポイントファイルを指定しています。
この他にも使用するメモリ量やCPUコア数などを指定できます。

"#"で始まる行のブロック(2-3行目)は、ルートセクションとよばれます。
ここでは実行するジョブのタイプを指定します。
ルートセクションは空行(1行空ける)まで続きます。

4行目はタイトルセクションです。
コメントを記述します。
タイトルセクションは2行以内です。
タイトルセクションの終了には空行が必要です。

7行目からは分子指定セクションが始まります。
はじめの"0 1"は、それぞれ分子の電荷とスピン多重度を表しています。
スピン多重度は以下の式で得られます。

.. math::

   2S+1

ここでSは分子のスピンの合計です。
スピンには :math:`\alpha` , :math:`\beta` がありますが、それぞれ :math:`\alpha = +1/2`, :math:`\beta = - 1/2` です。
したがって全ての軌道に電子が2つずつ入った閉殻系では、 :math:`S=0` となり、スピン多重度は1となります。

8行目からは分子の3次元座標を入力します。
この例ではカーテシアン座標での入力を示しています。
それぞれの行は一つの原子を示しています。
原子種、X, Y, Z座標の順に入力しています。

計算終了の確認
=======================================================================

Gaussianの正常終了
-----------------------------------------------------------------------

Gaussianが正常終了した場合は、最後に以下のような出力があるはずです。 ::
   
   Job cpu time:       0 days  0 hours  0 minutes  0.8 seconds.
   Elapsed time:       0 days  0 hours  0 minutes  2.7 seconds.
   File lengths (MBytes):  RWF=      6 Int=      0 D2E=      0 Chk=      1 Scr=      1
   Normal termination of Gaussian 16 at Thu Nov  1 04:43:37 2018.
